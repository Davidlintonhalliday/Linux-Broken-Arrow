What this is

Broken Arrow Linux is built from the LFS approach: understand the system, compile it, and own every part of it. This site tracks the build, decisions, and what I’m polishing next.

Documentation

The goal is simple docs that a normal human can follow. If you’re new, start with LFS Link, then check Downloads.

Roadmap

Bootable base → toolchain confidence → desktop choice → installers/scripts → “polish over time”. This will stay practical, not perfect.

Why “Broken Arrow”

Because the build is honest: you break things, fix them, learn the truth, and keep moving. The aim is to help others do the same.

“Do it once. Do it right. And when it breaks… rebuild it better.”
